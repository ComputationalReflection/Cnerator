Cnerator is a C source code generation tool. Generated programs can be compiled with any standard ANSI C compiler. The user may define different parameters such as the number of function to be generated or the probabilities of all the different syntactic constructs.

Here, you will find developer information about the implementation. However, if you are looking for
a `manual <https://github.com/ComputationalReflection/Cnerator/documentation>`_ about how to use Cnerator, 
or just for `getting started <https://github.com/ComputationalReflection/Cnerator>`_ information.
