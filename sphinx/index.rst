`Cnerator <https://github.com/ComputationalReflection/Cnerator>`_ is a C source code generation tool.
Generated programs can be compiled with any standard ANSI C compiler.
The user may define different parameters such as the number of function to be generated or the probabilities
of all the different syntactic constructs.
