run package¶

This package provides different example pre-configured executions of Cnerator.

Submodules¶

run.1-function-each-type module¶

Example execution of Cnerator to create a particular number of functions. It creates a program using the 1-function-each-type-10.json file, running different visitors.

run.15-function-each-type module¶

Example execution of Cnerator to create a particular number of functions. It creates a program using the 15-function-each-type.json file, running different visitors.

run.2000-function-void-no-void module¶

Example execution of Cnerator to create a particular number of functions. It creates a program using the 2000-function-void-no-void.json file, running different visitors.

run.300-function-each-type module¶

Example execution of Cnerator to create a particular number of functions. It creates a program using the 300-function-each-type.json file, running different visitors.

run.article module¶

Example execution of Cnerator to create particular functions. It creates a program using the article.json file, running different visitors.

run.defaults module¶

Example execution of Cnerator passing no parameters. It creates a random program considering the default probabilities.

run.defaults_one_file module¶

Example execution of Cnerator passing no parameters but generation in one file (-n 1).

run.help module¶

Example execution of Cnerator passing the --help parameter.

run.probs module¶

Example execution of Cnerator passing the -p parameter to specify the probabilities of two language constructs.

run.probs_file module¶

Example execution of Cnerator specifying probabilities in a json file (example_probs.json).
